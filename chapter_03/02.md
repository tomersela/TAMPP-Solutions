If the the memory is quiescently consistent, the individual registers are not necessarily quiescently consistent.

Counter example:

Let A and B run the following operations over time:
```
A: |---[ w(r1=0)  ]------[ r(r2=2)  ]--[ w(r1=1)   ]------
B: |---[ w(r2=0)  ]--[ r(r1=1)]--[ w(r2=2)]---------------
```

There's one quiescent period between operations w(r1=0), w(r2=0) to the other operations - r(r2=2), w(r1=1), r(r1=1), w(r2=2).
A valid total order can be:

```
A: w(r1=0) -> B: w(r2=0) -> A: w(r1=1) -> B: r(r1=1) -> B: w(r2=2) -> A: r(r2=2)
```

But H|r1:
```
A: |---[ w(r1=0)  ]------------------[ w(r1=1)   ]--------
B: |-----------------[ r(r1=1)]---------------------------
```

Now there's quiescent periods that force:
```
A: w(r1=0) -> B: r(r1=1) -> A: w(r1=1)
```
which force and invalid total order.

Same can be shown for H|r2.
