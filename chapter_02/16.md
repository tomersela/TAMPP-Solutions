### Claim 1. At most one thread gets the value `STOP`.

**Proof:**
Suppose, by way of contradiction, that two distinct threads $A$ and $B$ both return `STOP`.

A thread returns `STOP` only if:

1. it reads `goRight == false` at line 11,
2. it executes `goRight = true` at line 13,
3. it then reads `last == i` at line 14.

So both $A$ and $B$ must have passed line 11 while `goRight` was still `false`.

After line 13, neither thread writes `last`; line 14 only reads it. Therefore, when $A$ and $B$ evaluate line 14, the variable `last` has one single value.

For $A$ to return `STOP`, `last` must equal $A$.
For $B$ to return `STOP`, `last` must equal $B$.

Since $A \neq B$, this is impossible. Contradiction.

Therefore, at most one thread gets the value `STOP`.


### Claim 2. At most $n - 1$ threads get the value `DOWN`.

**Proof:**
A thread returns `DOWN` only if:

1. it reads `goRight == false` at line 11,
2. it executes line 13,
3. it then reads `last != i` at line 14.

Let $S$ be the set of threads that pass line 11, that is, the threads that read `goRight == false`.

Among the threads in $S$, consider the one that writes to `last` last at line 10. Call that thread $A$.

Since $A \in S$, thread $A$ also reaches line 14. After $A$'s write to `last`, no thread in $S$ writes `last` again. Also, any thread that reads `goRight == true` returns immediately at line 12 and does not write `last`.

Hence, when $A$ executes line 14, we have `last == A`, so $A$ does **not** return `DOWN`; it returns `STOP`.

Therefore not all threads in $S$ can return `DOWN`. At most $|S| - 1$ of them do. Since $|S| \le n$, at most $n - 1$ threads get the value `DOWN`.


### Claim 3. At most $n - 1$ threads get the value `RIGHT`.

**Proof:**
A thread returns `RIGHT` only if it reads `goRight == true` at line 11.

Initially, `goRight == false`. Therefore, before any thread can return `RIGHT`, some thread must first execute line 13 and set `goRight = true`.

Let $A$ be the first thread to execute line 13. Since $A$ reached line 13, it must have read `goRight == false` at line 11. Therefore $A$ does **not** return `RIGHT`; instead, it continues to line 14 and returns either `STOP` or `DOWN`.

So at least one thread does not get the value `RIGHT`.

Therefore, at most $n - 1$ threads get the value `RIGHT`.
